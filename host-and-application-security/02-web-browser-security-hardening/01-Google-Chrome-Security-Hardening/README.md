## Overview

This lab demonstrates how to **review and configure security and privacy settings in Google Chrome** to reduce security risks and improve user privacy.

In this lab, I performed a security review of Chrome’s built-in protections and configured settings to help protect against:

- Credential compromise
- Malicious websites
- Tracking
- Unsafe downloads
- Insecure connections
- Pop-ups and malicious redirects
