## Overview

This lab demonstrates how to **review and configure security and privacy settings in Google Chrome** to reduce security risks and improve user privacy.

In this lab, I performed a security review of Chrome’s built-in protections and configured settings to help protect against:

- Credential compromise
- Malicious websites
- Tracking
- Unsafe downloads
- Insecure connections
- Pop-ups and malicious redirects

# Security Areas Covered

This lab focuses on the following browser security controls:

### Credential Protection

- Password breach detection
- Weak password identification
- Secure password management

### Privacy Protection

- Blocking third-party cookies
- Managing site permissions
- Reviewing stored site data

### Secure Browsing

- Enhanced Safe Browsing protection
- Detection of malicious websites

### Secure Connections

- Enforcing HTTPS connections
- Warning users about insecure sites

### Browser Hardening

- Pop-up and redirect blocking
- Autofill data management
- Payment and address autofill security
