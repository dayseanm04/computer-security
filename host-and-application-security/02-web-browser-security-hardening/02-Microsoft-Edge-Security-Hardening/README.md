## Overview

This lab demonstrates the process of reviewing and configuring **Microsoft Edge security and privacy settings** to improve browser security and reduce exposure to common web-based threats.

Web browsers are frequently targeted by attackers because they interact with untrusted websites, downloads, and external content. Proper browser configuration helps mitigate risks such as:

- Credential compromise
- Malicious websites
- Tracking technologies
- Unsafe downloads
- Data exposure through browser storage

This lab focuses on applying **browser hardening techniques** to strengthen Microsoft Edge's built-in security controls.

# Security Areas Covered

This lab focuses on the following Microsoft Edge security features.

### Credential Protection
- Password manager review
- Weak and reused password detection
- Secure password storage

### Privacy Protection
- Tracking prevention
- Cookie management
- Site permission review

### Browser Security

- Microsoft Defender SmartScreen protection
- Malicious site detection
- Secure browsing configurations

### Data Protection

- Autofill data management
- Browser data clearing
- Control of stored site information
