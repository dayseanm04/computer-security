# 01-Guide

This folder contains the step-by-step walkthrough for the **OpenPuff Steganography Data Hiding** lab/project. The guide is split into three parts, follow them in order.

---


## 📂 Files

| File | Description |
|------|-------------|
| [**`01-download-and-setup-OpenPuff.md`**](./01-download-and-setup-OpenPuff.md) | Download OpenPuff, create the secret message, set up folders, and launch the app |
| [**`02-hide-message-with-OpenPuff.md`**](./02-hide-message-with-OpenPuff.md) | Set passwords, load the message and carrier file, and embed the hidden message |
| [**`03-unhide-message-with-OpenPuff.md`**](./03-unhide-message-with-OpenPuff.md) | Recover the hidden message from the carrier file using the Unhide feature |
