## Overview

In this lab/project I performed a security assessment of SSL/TLS protocol implementation for secure web communications. The analysis evaluates both the server-side TLS configuration and the client-side browser validation process.

The servers were analyzed using the Qualys SSL Labs testing platform to review TLS protocol support, cipher suites, certificate configuration, and overall cryptographic strength. I conducted Client-side testing using Google Chrome and Microsoft Edge to analyze certificate validation, TLS handshake details, and encryption parameters.

The results provide insight into how secure HTTPS connections are established and how browsers verify the trustworthiness of web servers.

---

## Objectives

- Analyze the SSL/TLS configuration of a web server
- Inspect certificate validation and connection security using web browsers
- Identify potential weaknesses in TLS/SSL configurations
- Document the testing process and security findings
